Works for Firefox (not tested for others)
1. Go to about:config
2. Set xpinstall.signatures.required to false   

Create a file in downloads.
Git clone the repository into the file

Go to 
about:debugging#/runtime/this-firefox
Go to temporary addons, upload the manifest.json provided in the file. If successful, you will see a clock icon.

3. click on the add-on at top right of toolbar, click manage, and pin on toolbar
4. Click on the clock icon to start
